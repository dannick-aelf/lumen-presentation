## Quick Reference

### Key Metrics
- **Market Size**: $21.5B TAM
- **Growth Rate**: 22% CAGR
- **Target Users**: Gen Z & Millennials (18-40)
- **Year 1 Target**: 500K users, 5% conversion
- **Year 3 Target**: 2.5M+ users, 15% market share, $50M ARR

### Core Features
1. **Alignment** (Primary) - Daily check-ins, pattern-based insights
2. **Goal Tracking** (Primary) - Pattern-aligned goal achievement
3. **Friend Compatibility** (Primary) - Social wellness features

### Design Principles
- Mobile-first, viewport-responsive
- Dark theme (black/gray-900)
- Gradient cards for visual interest
- Smooth animations (60fps)
- WCAG 2.1 AA accessibility

### Positioning
- **Old**: Astrology (App Store rejected)
- **New**: AI-Powered Alignment & Wellness Platform (App Store compliant)
